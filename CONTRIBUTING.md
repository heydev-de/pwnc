# No Contributions

Thank you for your interest! This repository provides a rolling release snapshot of the PWNC installation package.

At this time, the project is not open for contributions, and I am not accepting pull requests.

## How to Engage

- **Report Issues:** If you find a bug, please [report it here](https://github.com/heydev-de/pwnc/issues/new).
- **Suggestions:** If you have suggestions or feature requests, please [share them here](https://github.com/heydev-de/pwnc/discussions/categories/ideas).
