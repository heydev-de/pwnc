## SHA-256 Hashing Utility

Pure JavaScript implementation of the SHA-256 cryptographic hash function. This utility provides a client-side hashing mechanism for strings, returning a 64-character hexadecimal digest. It is used for data integrity checks, password hashing (when combined with salts), and other cryptographic operations within the PWNC Web Platform.

---

### `sha256(str)`

#### **Purpose**
Computes the SHA-256 hash of the input string. The function processes the string according to the SHA-256 specification, including padding, message scheduling, and compression.

#### **Parameters**

| Name | Type     | Description                                                                 |
|------|----------|-----------------------------------------------------------------------------|
| str  | string   | The input string to be hashed. Must be a valid UTF-16 string (no multibyte). |

#### **Return Value**

| Type   | Description                                                                 |
|--------|-----------------------------------------------------------------------------|
| string | A 64-character lowercase hexadecimal string representing the SHA-256 hash. |

#### **Inner Mechanisms**
1. **Padding**: Appends a single `0x80` byte, followed by null bytes (`\0`) to align the message to a 56-byte boundary within a 64-byte block.
2. **Length Encoding**: Appends the original message length (in bits) as a 64-bit big-endian integer.
3. **Message Schedule**: Expands the 16-word block into 64 words using bitwise rotations and XOR operations.
4. **Compression**: Processes each 64-word block through 64 rounds of hashing, updating the 8-word hash state.
5. **Output**: Converts the final 8-word hash state into a hexadecimal string.

#### **Usage Context**
- Client-side password hashing before transmission (e.g., for secure login without sending plaintext).
- Generating unique identifiers for files or data blobs.
- Verifying data integrity in browser-based applications.

#### **Example**
```javascript
// Hash a password before sending to the server
const password = "user_password_123";
const hashedPassword = sha256(password);
// Send hashedPassword to the server for authentication
console.log(hashedPassword); // e.g., "ef92b778bafe771e89245b89ecbc08a44a4e166c06659911881f383d4473e94f"
```

> **Note**: This implementation assumes the input string contains only ASCII-compatible characters (no multibyte UTF-8). For multibyte strings, pre-encode the string to UTF-8 (e.g., using `TextEncoder`) before hashing.
